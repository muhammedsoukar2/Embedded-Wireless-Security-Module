🔷 Embedded Wireless Security Module

I designed and developed the schematic and PCB of a compact embedded system based on STM32 and ESP32.

The system integrates sensor interfacing, wireless communication, power management, and control infrastructure on a single 4-layer PCB, providing a solution for industrial monitoring and IoT-based security applications.

Within the scope of the project:
• 4-layer PCB design (Top/Bottom/Power/Ground)
 • Dual-processor architecture (STM32-based control and ESP32 for wireless communication)
 • Power management (220V AC input with 12V, 5V, and 3.3V DC conversion)
 • Motion and temperature sensing with signal processing
 • Relay and buzzer-based alert mechanisms
 • Communication interfaces: UART, SPI, and I2C
 • Schematic design, footprint, and library creation
 • Decoupling strategies and power integrity improvements
